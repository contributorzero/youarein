Page Structure

TagWhat it doesNote
<!DOCTYPE html>Tells browser this is HTML5Always the first line
<html>Root of the whole pageWraps everything
<head>Info about the page — not visibleTitle and CSS links go here
<title>Text shown in browser tabOnly one per page
<body>Everything visible on the pageOnly one per page
<link>Connects external CSS fileGoes inside head
<style>CSS written directly in HTMLGoes inside head
<script>JavaScript written in HTMLUsually before closing body
<meta>Page info for browsers and Googlecharset, viewport, description

Text and Headings

TagWhat it doesNote
<h1>Biggest headingOnly one per page
<h2> to <h6>Smaller headingsh6 is smallest
<p>Paragraph — starts a new lineMost used text tag
<span>Inline container for styling textLike div but stays inline
<strong>Bold textlike this
<em>Italic textlike this
<br>Line breakNo closing tag needed
<hr>Horizontal divider lineNo closing tag needed
<blockquote>Indented quote blockFor long quotes
<code>Monospace font for codelike this
